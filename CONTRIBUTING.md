# Contributing

Add a tool, guide, or tip that genuinely helps someone run Claude Code from China.

## Rules
1. **One item per PR.**
2. **Real and useful** — no marketing filler, no dead links.
3. **Disclose affiliation** if you made it. You'll still be listed.
4. **No affiliate/referral links.**
5. **Competitors welcome on equal terms** — a better tool than ours belongs here.
6. Put it in the right section; keep the one-line description factual.

## What gets rejected
- SEO spam / listings with no catalogue value.
- Anything that only works by taking a vendor's "no downgrade" word for it — link a way to *verify* instead.
